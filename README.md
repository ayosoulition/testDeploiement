# 🍽️ IGMAAA

A simple and modern web application for digital restaurant menus and ordering.

🌐 Live Demo: https://igmaaa.netlify.app/

---

## 📌 About the Project

IGMAAA is a lightweight web application designed to digitalize restaurant menus and improve the ordering experience.

It allows customers to browse menu items easily and helps restaurants manage their offerings in a more modern and efficient way.

---

## ✨ Features

- 📱 Responsive design (mobile & desktop)
- 🍔 Digital menu browsing
- 🧭 Easy navigation between categories
- 🖼️ Visual display of products/items
- ⚡ Fast and lightweight UI
- 🛒 Simple ordering experience
